
⭐ คุณสมบัติของเว็บไซต์ (Website Features)

อ้างอิงจากโค้ดในไฟล์ `โปรเจคเว็ปไซต์อะไหล่รถยนต์.html` 

1. ระบบค้นหาสินค้า (Search Function)
* มีช่องค้นหาสินค้าแบบ Real-time
* ค้นหาได้จาก *ชื่อสินค้า / รุ่นรถ / รหัสสินค้า / แบรนด์*
* ระบบจะกรองสินค้าแบบ dynamic ทันทีเมื่อพิมพ์

2. ระบบหมวดหมู่สินค้า (Category Filter)
* คลิกเลือกหมวดหมู่เพื่อดูสินค้าที่เกี่ยวข้อง
* เช่น ผ้าเบรก, แบตเตอรี่, ไส้กรอง, โช้คอัพ ฯลฯ

3. ระบบแสดงสินค้า (Product Listing)
* แต่ละสินค้าแสดง:
  * รูปสินค้า
  * ราคา
  * ชื่อสินค้า
  * รหัสสินค้า
  * แบรนด์
  * จำนวนสต็อก
* มีปุ่ม **ใส่ตะกร้า**
* มีปุ่ม **ดูรายละเอียดสินค้า**


4. หน้ารายละเอียดสินค้า (Product Detail Modal)
* คลิกดูข้อมูลแบบ Popup Modal
* แสดงภาพใหญ่ รายละเอียด ราคา รหัสสินค้า สต็อก

5. ระบบตะกร้าสินค้า (Shopping Cart System)
* แสดงรายการสินค้าในตะกร้า
* เพิ่มจำนวนสินค้า / ลดจำนวนสินค้า / ลบออก
* คำนวณยอดรวมอัตโนมัติ
* ระบบส่วนลด เช่น ส่วนลด 10% เมื่อยอดถึงกำหนด
* แสดงยอดหลังลดและยอดสุทธิ

6. ระบบชำระเงิน (Payment Selection)
* ตัวเลือกช่องทางชำระเงิน:

  * พร้อมเพย์
  * QR Code
  * TrueMoney Wallet
* ระบบแสดงยอดสุทธิ พร้อมส่วนลด
* Modal ขึ้นแจ้งผลและปิดการชำระเงิน

7. UI/UX ดีไซน์แบบ Responsive
* รองรับมือถือ แท็บเล็ต เดสก์ท็อป
* เลเอาต์ยืดหยุ่นตามความกว้างหน้าจอ
* Navigation ซ่อนอัตโนมัติเมื่อจอแคบ



 ⭐หลักการทำงานของเว็บไซต์ (How the System Works)

1. โครงสร้าง HTML + CSS
* ใช้ HTML/CSS ในการสร้างหน้าเว็บ
* ออกแบบด้วย Grid Layout + Flexbox
* ใช้โทนสีตามแบรนด์ Ironparts (แดง–เทา)

2. การสร้างสินค้า (Product Data)
```js
const products = [ ... ];
```

* ข้อมูลสินค้าเก็บไว้ใน Array
* ใช้ JavaScript นำข้อมูลมาแสดงบนเว็บผ่านฟังก์ชัน `renderProducts()`

3. ระบบค้นหา (Search)
* ใช้ event `oninput`
* เมื่อผู้ใช้พิมพ์ → ฟิลเตอร์ Array ของสินค้า
* แสดงผลลัพธ์ทันทีแบบไม่ต้องโหลดหน้าใหม่

4. ระบบตะกร้า (Cart Logic)
* ตะกร้าคือ Array ที่เก็บสินค้าที่เพิ่มลงมา
* ปรับจำนวน (qty++) / ลบสินค้า / คำนวณราคา
* ระบบคำนวณส่วนลดเงื่อนไข
* แสดงผลใน Modal ด้านขวา

5. Modal รายละเอียดสินค้า & Modal ชำระเงิน
* ทำงานด้วยการสลับ class `show` เพื่อเปิด/ปิด
* แสดงข้อมูลที่ดึงจาก Array ของสินค้า

6. ระบบชำระเงิน
* คำนวณยอดรวม → ส่วนลด → ยอดสุทธิ
* แจ้งเตือนด้วย alert (Mock-up)
* รีเซ็ตตะกร้าหลังชำระเงินเสร็จ


⭐ วิธีใช้งานเว็บไซต์ (User Flow)

เขียนให้เข้าใจง่ายตามขั้นตอนจริงของผู้ใช้งาน

ขั้นตอนที่ 1 : ค้นหาสินค้า
1. พิมพ์ชื่อสินค้า / รุ่นรถ / แบรนด์ / รหัสสินค้า
2. ระบบจะกรองรายการสินค้าให้อัตโนมัติ
3. หรือเลือกผ่านหมวดหมู่ด้านซ้าย


ขั้นตอนที่ 2 : เลือกสินค้า
1. ดูภาพสินค้าและราคา
2. หากต้องการดูรายละเอียดเพิ่มเติม → คลิก “ดูรายละเอียด”
3. หากต้องการซื้อ → คลิก “ใส่ตะกร้า”


ขั้นตอนที่ 3 : จัดการตะกร้าสินค้า
ในหน้าตะกร้าผู้ใช้สามารถ:

* เพิ่มจำนวนสินค้า
* ลดจำนวนสินค้า
* ลบสินค้าออก
* ดูยอดรวม → ส่วนลด → ยอดสุทธิ

เมื่อพร้อม → กดปุ่ม **“ชำระเงิน”**

ขั้นตอนที่ 4 : เลือกวิธีชำระเงิน
1. เลือกช่องทางชำระเงิน:

   * พร้อมเพย์
   * QR Code
   * TrueMoney Wallet
2. ระบบจะแสดงข้อมูลยอดชำระ
3. หลังยืนยัน ระบบจำลองการชำระเงินและล้างตะกร้า
